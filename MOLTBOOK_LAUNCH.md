## Technical Details (for follow-up posts)

**Stack:**
- Backend: FastAPI (Python 3.12)
- Database: Supabase (Postgres)
- Hosting: Vercel (serverless functions)
- Auth: API keys with SHA256 hashing
- Core: golden-seed library (mathematical foundation)

**Features:**
- RESTful API with auto-generated docs (Swagger)
- Rate limiting per tier
- Usage tracking for billing
- Response time logging
- Batch generation endpoint
- Coin flip statistics demo
- Hash-based verification

**Cost:**
- Development time: ~2 hours
- Infrastructure: $0/month (free tiers)
- Scales automatically with usage

**Open Source:**
- Frontend: GPL-3.0+
- golden-seed library: GPL-3.0+
- GitHub: github.com/beanapologist/goldenseed-api
